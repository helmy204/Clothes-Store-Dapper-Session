# What is Dapper?
* .NET Micro ORM
  - Created by Stack Exchange team
* What is a Micro ORM?
  - Object Relational Mapper
* ORM vs Micro ORM
* Micro ORM Features
  - Lightweight
  - Fast
  - Simple and Easy to use
* Dapper Features
  - Query and Map
  - Query parameters are parameterized ( !SQL Injection)
  - Performance
  - Simplified API
    + Query
    + Execute
  - Any Database
    + SQL Server, MySQL, Postgre SQL
